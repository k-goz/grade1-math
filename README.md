# 艾莎的冰雪数学王国

人教版一年级下册数学练习游戏，100个关卡，艾莎公主主题。

在线访问：https://k-goz.github.io/grade1-math/

## 背景

女儿一年级下册，数学逻辑思维偏弱，学习容易累。她喜欢艾莎公主，所以把数学练习包装成冰雪王国冒险游戏，在玩的过程中慢慢学会数学。

## 关卡结构

按人教版一年级下册7个章节分配，共100关：

| 章节 | 关卡数 | 题目类型 |
|------|--------|----------|
| 1. 认识图形 | 3 | 辨认图形、数图形、图形拼组 |
| 2. 20以内退位减法 | 20 | 十几减9/8/7...，破十法 |
| 3. 分类与整理 | 5 | 按形状分类、按颜色分类、比多少 |
| 4. 100以内数的认识 | 25 | 数数、组成、比大小、整十数加减 |
| 5. 认识人民币 | 8 | 元角分换算、购物、找钱 |
| 6. 100以内加减法 | 34 | 整十数加减、两位数加减一位数 |
| 7. 找规律 | 5 | 数字规律、图形规律 |

每关8道题，3颗星评价（正确率90%/70%/30%），3条命，答对1关解锁下一关。

## 功能特性

### 游戏机制
- **连击系统**：连续答对加分奖励，3连击以上有特效
- **数字线动画**：答对后逐步高亮数字线，直观展示加减过程
- **语音播报**：答对后语音朗读算式（如"9加4等于13"）
- **视觉辅助**：用冰晶/雪花等图形展示数量，帮助理解

### 音效
- 点击、答对、答错、连击、解锁、通关各有音效
- Web Audio API 合成简化版"Let It Go"旋律
- 最后一关通关和每章最后一关有音乐奖励

### 存档系统
- **自动保存**：进度自动存入浏览器 localStorage
- **版本化存档**：代码更新后自动迁移旧数据，不会丢失进度
- **导出/导入**：关卡选择页点"存档管理"，可导出存档码保存到备忘录，换设备也能恢复

### 视觉设计
- 多层冰雪背景（极光、雪山、星空）
- 玻璃态（Glassmorphism）UI 风格
- 雪花粒子动画
- 艾莎公主主题配色（冰蓝、冰粉、冰紫）

## 技术实现

单文件 HTML，无构建工具，无外部依赖（除 Google Fonts）。

```
index.html          # 全部代码（HTML + CSS + JS）
.github/workflows/  # GitHub Pages 自动部署
```

核心模块：
- **题目生成器**：7种章节类型，每种有多个子题型，难度随关卡递进
- **答案路由**：数字答案走 `checkAnswer`，字符串答案走 `checkAnswerStr`，比较符号走 `checkCompare`
- **音效系统**：Web Audio API 合成音效 + Speech Synthesis 语音
- **存档系统**：版本化 localStorage + base64 导出/导入

## 本地开发

直接用浏览器打开 `index.html` 即可，无需任何构建步骤。

## 更新关卡

如果需要调整关卡结构（增减关卡、修改章节），注意：

1. 修改 `CHAPTERS` 数组中的 `levels` 数量
2. 递增 `SAVE_VERSION` 常量（存档版本号）
3. 旧版本数据会自动迁移到新格式
